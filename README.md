# New Tab Extension

Page de nouvel onglet minimaliste avec horloge et raccourcis.

## Utilisation rapide

1. Ouvre un nouvel onglet.
2. Clique sur `Ajouter` pour créer un raccourci (nom + URL + image optionnelle).
3. Clique sur l'icône en bas a droite pour afficher tous les raccourcis.
4. Fais un clic long sur un raccourci pour le modifier, le supprimer ou le deplacer.
5. Fais un clic long sur l'horloge pour configurer le style.

## Installation locale (Firefox)

1. Ouvre `about:debugging#/runtime/this-firefox`.
2. Clique sur **Charger un module complementaire temporaire**.
3. Selectionne `manifest.json`.

## Notes

- Les donnees sont sauvegardees en local dans le navigateur.
